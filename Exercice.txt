Exercice avec Python: 
Creer un systeme de gestion bancaire qui aura des :
Des utilisateur comme  : client et eun admin

*Role du client est de:
- Faire un dépot 
- Faire Retrait
- Faire Transfert
- Consulter son solde
- consulter ses historique pour chaque client

*Role Admin est de:
- Creer des comptes
- Bloquer des comptes
- Voir les historiques des transactions

*Pour la realisation voici les Conceptes à utiliser :
- les Classes
- Les objets ou instances
- Encapsulation
- Les classe abstrait
- Heritages

*Pour la Base de Donnée on utilise les Fichiers
